Machine Learning이란 무엇이냐

머신러닝이란
컴퓨터가 스스로 학습하는 것,
사람이 Input data만 넣는 것이 아니라
Output data(결과값)까지 넣어주는 것
(특정한 프로그램 없이도 학습할 수 있도록 하는 것)

ex)자율주행차, 유튜브추천영상, 스팸메일함,질병식별

,표정, 감정, 목소리 등 인식
,이상현상 식별 (건강 상태, 신용카드 사기 거래, 이상 기온으로 인한 기계 오작동 식별) ,예상(주식, 부동산, 환율 예측)
